# Watch & Listen Studios website

This is the reorganised GitHub Pages project for watchandlistenstudios.com.

## Structure

- `index.html` — page content and structure
- `css/styles.css` — all website styling
- `js/script.js` — interactive behaviour and animations
- `images/` — hero, background, podcast artwork and Nicole Goodman portrait
- `CNAME` — GitHub Pages custom domain

## Publishing

Upload the **contents of this folder** to the root of the existing
`Watch-and-Listen-Studios` GitHub repository. Keep GitHub Pages set to:

- Branch: `main`
- Folder: `/(root)`

Do not upload only `index.html`; the CSS, JavaScript and image folders are required.
